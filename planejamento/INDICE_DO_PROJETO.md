# Indice do Projeto

Este diretorio organiza a base de planejamento para um sistema de biblioteca academica mais moderno e mais flexivel que o Pergamum.

## Objetivo

Criar uma plataforma completa para:

- catalogacao bibliografica e de exemplares
- circulacao, emprestimos, renovacoes e reservas
- aquisicoes, inventario, etiquetas e RFID
- automacoes de comunicacao e token de devolucao
- relatorios operacionais, gerenciais e analiticos
- portal do leitor, APIs e integracoes institucionais

## Decisao estrategica adotada

Para esta base inicial, o projeto foi estruturado como um sistema feito do zero, com arquitetura moderna, API-first e modular. Isso reduz dependencia de legados ruins e permite evoluir para uma solucao realmente superior, em vez de apenas copiar comportamentos antigos.
